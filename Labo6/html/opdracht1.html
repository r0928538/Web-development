Herschrijf dit fragment en de CSS-regels zonder classes op elk <li> element te moeten gebruiken:

Code html:
<ul>
<li class="menuitem"><a href="...">products</a></li>
<li class="menuitem"><a href="...">distributors</a></li>
<li class="menuitem"><a href="...">contact</a></li>
</ul>

Code css:
.menuitem {
display:inline-block;
width:150px;
}
.menuitem > a {
color:red;
}

Herschreven html:
<ul class="menuitem">
<li><a href="...">products</a></li>
<li><a href="...">distributors</a></li>
<li><a href="...">contact</a></li>
</ul>

Herschreven css:
.menuitem > li{
display:inline-block;
width:150px;
}
.menuitem > li > a{
color:red;
}

Wat er getoond wordt:

  • products
  • distributors
  • contact
