Opdracht 3:

De generator voorziet de cellen in deze rijen van een andere achtergrondkleur, zoek op waar dit precies staat in de gegenereerde CSS.
Hier: .datagrid table tbody .alt td { background: #B0B9F4; color: #000000; }

Waarom heeft de volgende CSS regel:
.onbekend {background-color:red;}
geen effect als je deze toepast op een <td> cel in een <tr> rij met class="alt", maar wel in een <tr> rij zonder deze 'alt' class?
Omdat de classe .alt een grotere specifiteit heeft, dus het heeft voorrang op de classe .onbekend. De oplossing is om de classe .onbekend meer specifiek te maken dan classe .alt.
Bv. op deze manier: .datagrid table tbody .alt td.onbekend, .onbekend{ background-color: lightcoral; }
